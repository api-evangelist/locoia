# Locoia (locoia)

Locoia is a low-code integration platform as a service (iPaaS) that enables businesses to integrate their entire tech stack and automate processes and reporting via a drag-and-drop workflow editor with 200+ pre-built connectors. Hosted in Frankfurt, Germany, with GDPR/DSGVO compliance and 99.99% uptime.
